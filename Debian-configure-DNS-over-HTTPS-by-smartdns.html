Debian 使用 SmartDNS 配置 DNS-over-HTTPS

本文类似 macOS 配置 DNS-over-HTTPS SmartDNS 简化版，配置适用于 Debian 系 Linux 发行版，Ubuntu 系统存在 53 端口占用问题。

SmartDNS 安装及配置

安装 SmartDNS：

sudo apt-get install smartdns

配置文件位于/etc/smartdns/目录。

SmartDNS 配置文件 /etc/smartdns/smartdns.conf 样例：

# 设置监听的DNS端口
bind [::]:53
bind-tcp [::]:53

tcp-idle-time 300

# 缓存大小与性能设置
cache-size 32768
prefetch-domain yes

# 持久化缓存设置(可选设置)
cache-persist yes 
cache-file /tmp/smartdns.cache

serve-expired-prefetch-time 21600
serve-expired yes
serve-expired-ttl 86400
serve-expired-reply-ttl 30
cache-checkpoint-time 86400

#禁用 SmartDNS 全局的测速设置
speed-check-mode none

force-qtype-SOA 65

#设置TTL最小值和最大值
rr-ttl 60
rr-ttl-min 0
rr-ttl-max 400

# 日志设置
log-level error
log-file /var/log/smartdns.log
log-size 5M
log-num 2

# 设置默认DNS服务器
# 设置国内上游DNS服务器
server-https https://dns.alidns.com/dns-query 
server-https https://doh.pub/dns-query 
server-https https://120.53.53.53/dns-query

server-tls dns.alidns.com
server-tls dot.pub

server-h3 h3://dns.alidns.com/dns-query

SmartDNS 去广告

下载广告过滤文件到/etc/smartdns目录：

wget https://anti-ad.net/anti-ad-for-smartdns.conf -O /etc/smartdns/anti-ad-smartdns.conf

可用的去广告列表：

  • anti-AD: https://anti-ad.net/anti-ad-for-smartdns.conf
  • adrules: https://adrules.top/smart-dns.conf
  • neodevhost: https://raw.githubusercontent.com/neodevpro/neodevhost/master/lite_smartdns.conf

在配置文件 /etc/smartdns/smartdns.conf 最后添加：

# 引入去广告配置文件
conf-file /etc/smartdns/anti-ad-for-smartdns.conf
conf-file /etc/smartdns/smart-dns.conf

SmartDNS 添加黑名单

需要手动添加黑名单，新建blacklist.txt，格式如下：

2345.com
mairuan.com
hao123.com

在配置文件 /etc/smartdns/smartdns.conf 最后添加：

# 引入黑名单
domain-set -name blacklist -file /etc/smartdns/blacklist.txt
address /domain-set:blacklist/#

SmartDNS 服务管理

启动 SmartDNS 服务：

sudo systemctl start smartdns
sudo systemctl enable smartdns

查看 SmartDNS 状态：

sudo systemctl status smartdns

检查 SmartDNS 是否正常启动：

sudo lsof -Pni UDP:53

或者：

sudo netstat -lnup | grep smartdns

测试 DNS 解析：

dig @127.0.0.1 www.baidu.com

查看 SmartDNS 错误：

sudo journalctl -u smartdns
# or
sudo cat /var/log/smartdns.log

启动、重启与停止 SmartDNS 服务：

sudo systemctl start/restart/stop smartdns

每次修改 SmartDNS 配置文件都重启 SmartDNS 服务。
